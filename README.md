## Quick start

Quick start options:

- [Download from Github](https://github.com/VikNikolaiev/spa-comments-fe.git).
- Clone the repo: `git clone https://github.com/VikNikolaiev/spa-comments-be.git`.
- run `npm i`
- run `npm start`
